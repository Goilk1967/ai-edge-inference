# AI Edge Inference ⚡

C++ library for optimized neural network inference on edge devices, supporting SIMD acceleration and hardware-specific optimizations.

## 🌟 Key Features
- **SIMD Acceleration**: Optimized for ARM Neon and x86 AVX instructions.
- **Quantization Support**: Native 8-bit and 4-bit integer quantization for reduced memory footprint.
- **Cross-Platform**: Built for Linux, Android, and RTOS.

## 🛠️ Build
```bash
mkdir build && cd build
cmake ..
make
```
